* Org Me Up!
** Where am I?

This is cliffnotes on org-roids. Learn about anything. Mostly nerdy anythings. Then, forget about everything, and refer back to your notes.

To start:

1. Fork this repo.
2. Add a book to the [[file+emacs:books/][books/]] folder (e.g. books/practical-common-lisp).
3. Link to your book in the /library/ section of this document.
4. Make sure to follow the hackneyed citation style, and tag your books for easy lookup.
5. Submit a pull request to get your book added to the main branch. Or, horde your learnings like a lemming.

** Some Useful Tags

|-----+-----------------------------------------------------|
| Tag | Meaning                                             |
|-----+-----------------------------------------------------|
| bab | Executable code snippets are included in the notes. |
| pdf | A manual.pdf file is included in the book folder    |
|-----+-----------------------------------------------------|
